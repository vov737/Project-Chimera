На русском: Это кодовый проект моего браузера под названием WindBrowser, должен выйти около зимы 2026 года. Функции раскрою позже, но скажу что там используется собственный движок рендеринга.

Лицензия: Apache-2.0

Кто делал: я (vov737) и Gemini 2.5 Flash :)

Сейчас, в исходниках репозитория находится версии 0.1.5 и 0.2.0 (в релизах будет 0.1.5, затем 0.2.0, их отличия системных ссылках браузера и в версии 0.2.0 есть пасхалка, найдите ее), требования: pip install pyfltk requests cryptography lxml js2py Pillow pydantic.
